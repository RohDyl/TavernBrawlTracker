⚡ Setup

Each player takes an edge. Deploy within 6" of your edge.
Place 3 Prize Tokens in the middle — at least 6" apart and 6" from each deployment zone.
Shuffle Gear deck. Each player draws an opening hand of 5 cards (flat). v9
Both roll d20. Highest wins Turn 1 initiative. Reroll ties.
Alternate placing models. Initiative player places first.
🍺 The Table Is the Terrain. All objects count as Clutter (Scavenge) and Cover (+4 Def). No terrain pieces needed.

🧠 Core Concepts

👊 Engaged

Within 1" of an enemy. Affects Retaliation & Fist Fights. May be engaged with multiple enemies.

👁 Line of Sight

Straight line attacker → target. Blocked by models & terrain taller than the model.

🛡 Cover

Any object between attacker & defender. +4 Defence. Same object = Cover & Scavenge.

🔄 Turn Structure — 4 turns or until one squad is fully KO'd

1 — Order Phase
Return all model cards to upright.
Gain 1 Order Token per model on the table.
Place a Command Token beside your BB (if alive).
Draw 1 Gear card per model on the table. v9
Bid initiative (T2+): both bid 1 card face-down, reveal, highest tip wins, both discarded. No bids = holder keeps. Tied = whoever didn't have it last turn.
2 — Action Phase
Initiative player chooses who activates first, then alternate.
1 Order Token = activate 1 model. Rotate card sideways.
Each activation: Move + 1 Action + 1 buff/Free card (see Actions). v9
Command Token: activates any model, even one already activated (once/model/turn unless Colonel). Lost if BB KO'd.
Pass when out of tokens or by choice. Phase ends when both pass.
3 — End Phase
1. Any unengaged model within 1" of a Prize claims it: +3 pts, remove token. v9
2. Check for victory.
3. Discard down to 7 cards. v9
Multiple friendly models on one prize — controlling player chooses who claims.

⚔️ Actions v9

⚡ Each activation: Move up to Speed + ONE Action + ONE buff/Free card. Action = Fist Fight OR play 1 Order card OR use BB ability. (Can't Fist Fight and play an Order card — they share the slot.)
⚔️ Move & Hit (default)
Move up to Speed, then Fist Fight an engaged enemy.
⚠️ Provokes Retaliation when moving while engaged.
🥊 Stand Your Ground
No movement → Fist Fight. Scrapper: +1 attack die. Brute: +1 defence die until next activation.
💨 Leg It
Move up to Speed twice (any directions). No Action this activation.
⚠️ Provokes Retaliation on each move while engaged.
🚶 Shove Through
Move Speed in a straight line. Path can't cross a model or terrain taller than 1".
✦ No Retaliation.
🎴 Scavenge — Free Action
Once/turn, a model touching Clutter draws 2 cards, keeps 1. Does NOT use your Action.

🥊 Combat v9

Every die: exactly the value = CRIT · natural 1 = auto-fail — on both attack & defence.
① Attack (must be engaged)

≤ Skill = Hit (1 dmg)

= Skill = CRIT (2 dmg, needs Crit Block)

Nat 1 = miss

② Defence

Roll 1 die per landed attack die + any bonus dice.

≤ Def = Block (stops 1 Hit)

= Def = CRIT BLOCK (stops 2 / cancels a Crit)

Nat 1 = fail

③ Resolve

Crit Block cancels a Crit (or 2 Hits). Normal Block can't touch a Crit. Net dmg = total unblocked.

🎯 Example

Brute rolls 14,9,1 → Crit + Hit. Scrapper (Def 10) rolls 2 dice: 10,6 → Crit Block + Block = 0 dmg!

👥 Gang Up v9

Scrappers +1 atk die per friendly Scrapper within 4" (max +2). Primary Fist Fight only — not free/reaction FFs.

⚔️ Retaliation

Move away / ranged card while engaged → each engaged enemy free FFs first (before action). Provoker KO'd = action cancelled. No Gang Up. No chaining.

Ranged (Gear cards): Need LoS. Can't shoot an enemy you're engaged with. Shooting elsewhere provokes Retaliation. +1 base die never applies to ranged. v9

🩸 Blood & Bloodied

Losing vs Spending

Losing — from attacks/cards.

Spending — using a Desperate Act.

Both reduce blood. At 0 → KO'd, removed.

🔥 Bloodied = Unlock v9

Scrapper ≤2 · Brute ≤3 · BB half or below (flip card).

When Bloodied: DAs use enhanced version & you may do 2 DAs per activation.
